## Binaries executable program

Please not that binary NUMERO6_FS is the version directly executable on Ubuntu Terminal 20.04 & 22.04
binary NUMERO6_PI is the binary file for Raspian PI3 & PI4
Just chmod +x and Run !!!

Binaires :

NUMERO6_FS = Ubuntu binary
NUMERO6_FS_PI = RaspBerry PI3 & PI4 binaries

i.e: chmod +x ./NUMERO6_FS 
then ./NUMERO6_FS to start a Tel session
